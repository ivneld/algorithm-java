## 문제
문자열 $S$가 주어졌을 때, 이 문자열이 어떤 소단위 문자열 $T$가 반복되어 만들어진 것인지 확인하고, 그 $T$의 최소 길이를 구하는 문제입니다.예시 1: S = "ababab" → "ab"가 3번 반복됨. 답: 2예시 2: S = "aaaaaa" → "a"가 6번 반복됨. 답: 1예시 3: S = "abcabc" → "abc"가 2번 반복됨. 답: 3예: S = "abcde" → 반복 없음(자기 자신). 답: 5
https://www.acmicpc.net/problem/1305
## 입력
첫째 줄에 광고판의 크기 L이 주어지고, 둘째 줄에 현재 광고판에 보이는 문자열이 주어진다.

## 출력
첫째 줄에 가능한 광고의 길이중 가장 짧은 것의 길이를 출력한다.

## 제한
- 1 ≤ L ≤ 1,000,000
- 광고판에 보이는 문자열은 알파벳 소문자로만 이루어져 있다.
