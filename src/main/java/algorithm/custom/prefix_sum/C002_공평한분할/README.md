## 문제 정보

- **플랫폼**: Custom
- **문제 번호**: C002
- **유형**: Prefix Sum
- **제목**: 공평한 분할

## 문제 설명

같은 길이의 두 배열 A, B를 위치 K에서 분할하면 4개의 구간이 생긴다.
네 구간의 합이 모두 같을 때 '공평'하다고 한다.
공평한 분할 방법의 수를 반환한다.

## 예시

```
A = [1, 2, 3, 4, 10]
B = [3, 0, 7, 0, 10]

K=4: [1,2,3,4] | [10]  → 합 10
     [3,0,7,0] | [10]  → 합 10  →  공평 ✓

반환값: 1
```

## 접근 방식

네 구간합이 모두 같으려면 아래 세 조건이 동시에 성립해야 한다.

```
2 * prefixA == totalA
2 * prefixB == totalB
prefixA == prefixB
```

K를 1부터 N-1까지 순회하며 누적합을 갱신하고 조건을 검사한다.

## 시간복잡도

O(n)

## 핵심 포인트

- 네 값이 모두 같다 = 각 배열의 절반 + 두 배열의 왼쪽 절반이 같다
- 전체 합이 홀수인 배열은 조건을 절대 만족할 수 없음 (2 * prefix == total 불가)
